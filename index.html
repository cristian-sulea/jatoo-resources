JaToo Resources

A free open source Java library created to ease the work with various resources (like locale specific texts, images, icons, etc).

Usage

ResourcesTexts texts = new ResourcesTexts(getClass());

System.out.println(texts.getText("title"));
System.out.println(texts.getText("description"));

ResourcesImages images = new ResourcesImages(getClass());

Image background = images.getImage("background.png");
ImageIcon icon = images.getImageIcon("icon-ok.png");

Please check the tutorial for more and detailed examples.
